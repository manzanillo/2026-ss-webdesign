Meilensteine der Web Geschichte

Meilensteine der Web Geschichte 1989 - Tim Berners-Lee reicht seinen Vorschlag Information Management: A Proposal bei CERN ein. 1990 - Erster Webserver und erster Webbrowser (WorldWideWeb) werden fertiggestellt. 1991 - Das Web wird öffentlich zugänglich gemacht, zunächst innerhalb der Wissenschaftsgemeinschaft. 1993 - Der Browser Mosaic erscheint und macht das Web erstmals einer breiten Öffentlichkeit zugänglich. 1994 - Gründung des World Wide Web Consortium (W3C) zur Standardisierung von Web-Technologien. 1995 - Kommerzieller Durchbruch: Unternehmen wie Amazon und eBay gehen online.

  • 1989: Tim Berners-Lee schlägt das Konzept des World Wide Web vor.
  • 1990: Entwicklung des ersten Webbrowsers und Webservers.
  • 1993: Veröffentlichung des Browsers Mosaic, der das Web für die breite Öffentlichkeit zugänglich macht.
  • 1994: Gründung des World Wide Web Consortium (W3C) zur Standardisierung von Web-Technologien.
  • 2000er: Aufkommen von Web 2.0 mit interaktiven und sozialen Plattformen.
